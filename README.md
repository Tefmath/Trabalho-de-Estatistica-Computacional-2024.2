# Trabalho-de-Estatistica-Computacional-2024.2
Trabalho feito na disciplina de Estatística Computacional na UFRJ em 2024.2

Professor Ralph dos Santos Silva

Grupo: Stefane Moraes

O trabalho propõe realizar inferência bayesiana para uma variável com distribuição Beta, usando uma amostra de dados. O objetivo é estimar os parâmetros α e β pelo método MAP (máximo a posteriori) e construir intervalos de credibilidade de 95%, assumindo distribuições a priori Gamma independentes, além de implementar tudo em R.
